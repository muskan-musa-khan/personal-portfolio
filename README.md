# Developer Portfolio

A personal portfolio website built with React and Tailwind CSS that presents projects, skills, and contact information in a clean and easy to explore layout. The site is designed as a single page application and adapts smoothly to both desktop and mobile screens. It allows visitors to browse work samples and send messages directly through an integrated contact form.

## Features

- Single page layout with clearly separated content sections  
- Fully responsive design for desktop and mobile devices  
- Project showcase section with live previews and descriptions  
- Contact form that lets visitors send messages directly from the site  
- Reusable UI components for consistent layout and styling  
- Clean and readable interface built with Tailwind CSS  

## Built With

- React  
- JavaScript  
- Tailwind CSS  

## Setup Instructions

### Clone the repository
```bash
git clone <your-repo-url>
```

### Move into the project folder
```bash
cd <project-folder-name>
```
### Install dependencies
```bash
npm install
```
### Start the development server
```bash
npm run dev
```
### Open your browser and visit:
```bash
https://localhost:5173
```
### To generate production files, run:
```bash
npm run build
```
